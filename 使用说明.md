# Life Coach AI 使用说明

## 🚀 快速开始

### 方法一：使用启动脚本（推荐）

#### macOS/Linux 用户：
```bash
# 进入项目目录
cd "第四课-life coach"

# 给启动脚本添加执行权限
chmod +x start.sh

# 运行启动脚本
./start.sh
```

#### Windows 用户：
```cmd
# 进入项目目录
cd "第四课-life coach"

# 双击运行或在命令行执行
start.bat
```

### 方法二：手动启动

```bash
# 进入项目目录
cd "第四课-life coach"

# 安装依赖
npm install

# 启动服务器
npm start
```

## 📱 访问应用

服务器启动成功后，在浏览器中打开：
- **应用首页**: http://localhost:3001
- **健康检查**: http://localhost:3001/health

## 💬 使用功能

### 基础对话
1. 在输入框中输入你想要咨询的问题
2. 点击"发送"按钮或按回车键
3. AI 助手会基于生活教练的专业知识为你提供建议

### 功能特点
- ✅ **智能回复**：基于 DeepSeek V3 的高质量 AI 对话
- ✅ **个性化建议**：针对生活、学习、工作等方面提供专业指导
- ✅ **实时对话**：流畅的实时聊天体验
- ✅ **响应式设计**：支持电脑、平板、手机等多种设备
- ✅ **历史记录**：保留对话历史，方便回顾
- ✅ **清空功能**：一键清空对话记录，重新开始

### 快捷操作
- **Enter 键**：快速发送消息
- **清空对话**：重置所有对话记录
- **状态指示**：实时显示连接和响应状态

## 🔧 故障排除

### 常见问题

#### 1. 端口被占用
**问题**：启动服务器时提示端口已被使用
**解决**：
- 修改 `server.js` 中的端口号（默认为 3001）
- 或者停止占用该端口的其他程序

#### 2. 依赖安装失败
**问题**：npm install 执行失败
**解决**：
```bash
# 清理缓存
npm cache clean --force

# 删除 node_modules 后重新安装
rm -rf node_modules package-lock.json
npm install
```

#### 3. API 连接失败
**问题**：前端无法连接到后端
**解决**：
- 确保服务器已成功启动
- 检查防火墙设置
- 验证 API Key 是否有效

#### 4. AI 无响应
**问题**：发送消息后 AI 不回复
**解决**：
- 检查网络连接
- 验证火山方舟 API Key 和配置
- 查看服务器控制台错误信息

### 调试模式
