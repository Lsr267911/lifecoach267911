# Life Coach AI - 快速开始指南

## 🎉 恭喜！你的 AI 生活助手已就绪

### 📱 立即体验
1. **打开浏览器访问**: http://localhost:3001
2. **在输入框中输入你的问题**
3. **按 Enter 或点击发送按钮**
4. **等待 AI 助手的回复**

### 💬 示例对话
试试这些话题来开始：

**关于学习成长：**
- "我如何提高学习效率？"
- "最近感觉学习动力不足，有什么建议吗？"

**关于职业发展：**
- "我想在职场上有更好的发展，应该从哪里开始？"
- "如何平衡工作压力和个人成长？"

**关于生活目标：**
- "我想要设定新的人生目标，有什么方法吗？"
- "如何培养更好的生活习惯？"

### 🔧 如果遇到问题

#### 1. 页面无法加载
```bash
# 检查服务器状态
curl http://localhost:3001/health

# 重启服务器
cd "第四课-life coach"
npm start
```

#### 2. AI 不回复
- 检查网络连接
- 刷新页面重试
- 查看浏览器控制台错误

#### 3. 想切换到模拟模式
```bash
cd "第四课-life coach"
pkill -f "node server.js"
MOCK_API=true npm start
```

### ✨ 功能特色

- 🤖 **智能对话**: 基于火山方舟 DeepSeek V3 的专业 AI
- 🎯 **个性化建议**: 针对你的具体情况提供定制化指导
- 💝 **温暖关怀**: 像朋友一样耐心倾听和建议
- 📱 **响应式设计**: 完美适配手机、平板、电脑
- 🔄 **持续对话**: 记住上下文，深入探讨问题
- 🎨 **现代界面**: 优雅的渐变设计和流畅动画

### 🚀 高级技巧

1. **详细描述问题** - 提供更多背景信息能获得更精准的建议
2. **多轮对话** - 通过连续提问深入探讨话题
3. **开放心态** - 尝试 AI 提供的新思路和方法
4. **定期回顾** - 查看对话历史，跟踪自己的成长
